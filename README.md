# W-Seminar-Arbeit: Modellierung und Implementierung einer dezentralen Spendenplattform

# Installation

1. Repository klonen:
    ```bash
    git clone https://github.com/Cetherium/W-Seminar-Arbeit.git
    ```

2. In das Verzeichnis wechseln:
    ```bash
    cd W-Seminar-Arbeit
    ```

3. ssh-server-1 starten:
    ```bash
    ssh root@vps-1-ip
    ./start-backend.sh
    ```

4. ssh-server-2 starten:
    ```bash
    ssh root@vps-2-ip
    ./start-backend.sh
    ```
